# Instant Search

Instant Search is a fast, offline, Apple-style search workspace for notes and files, built as a single static HTML app.

## Capabilities
- Drag-and-drop or pick files to ingest (`.txt`, `.md`, `.pdf`, `.png`, `.jpg`, `.jpeg`, `.webp`)
- Extract and index PDF text using PDF.js
- Create and edit rich-text notes in an inline editor
- Organize files and notes into nested folders with drag-and-drop
- Instant search with highlights and result preview
- Search modes: cluster (nearby terms), phrase, and boolean (`AND`, `OR`, `NOT`)
- Filter by content type and sort by relevance/newest/oldest
- Local persistence via `localStorage`
